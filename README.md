# Descripción General

Este programa asíncrono está diseñado para desarrollar competencias en el uso estratégico de herramientas de IA para desarrollo de software, desde fundamentos hasta implementación de workflows agénticos avanzados.

El programa está estructurado en 4 niveles progresivos, cada uno con tutoriales, labs prácticos y criterios de evaluación.

---

## 🎯 Estructura del Programa

### Nivel 1: Copilot Consciente
**Fundamentos de uso estratégico y seguro de Copilot en VS Code**

### Nivel 2: IA Multitarea
**Dominio de contexto multi-archivo y desarrollo simultáneo**

### Nivel 3: Workflows Agénticos
**Implementación de workflows Plan-Act-Observe-Reflect**

### Nivel 4: Agentes Especializados
**Uso de agentes especializados para tareas específicas**
