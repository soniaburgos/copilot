# Lab 3.4: Onboarding en Código Legacy (Análisis y Documentación Asistida)

## Objetivo

Usar IA para explorar y comprender código legacy, generando documentación y guías de onboarding.

## Duración: 2-3 horas

## Workflow Agéntico

### PLAN: Exploración
- Identificar módulos clave
- Planear análisis sistemático
- Definir qué documentar

### ACT: Análisis asistido
- Explorar código con Agente Onboarding
- Generar documentación automática
- Identificar patrones y flujos

### OBSERVE: Validación
- Verificar documentación es completa
- Validar comprensión

### REFLECT: Refinamiento
- Mejorar documentación
- Agregar ejemplos prácticos

---

**Entregables**: Guía de onboarding completa, documentación de arquitectura

**Versión**: 1.0
